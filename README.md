
# QuarksDarker-Splashscreen
This Is A Custom Splashscreen for KDE Plasma made by [IGFLAVIUS](https://store.kde.org/u/igflavius) on KDE Store. This REPO is used by me in a automated customization script.



<h1>Quarks Splash Darker</h1>

<h3>This splash is based on [Quarks Splash Dark](https://store.kde.org/p/1304256/) by ADHE</h3>

<h5>Clone This Repo And `cd` into it.</h5>

```
git clone https://github.com/googIyEYES/QuarksDarker-Splashscreen.git && cd QuarksDarker-Splashscreen
```

<h5>Install</h5>

```
kpackagetool5 -t Plasma/LookAndFeel -i quarkssplashdarker.tar.gz
```

<h5>Remove</h5>

```
kpackagetool5 -t Plasma/LookAndFeel -r quarkssplashdarker
```
